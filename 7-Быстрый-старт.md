БЫСТРЫЙ СТАРТ - ПОЛНАЯ ИНСТРУКЦИЯ ПО СОЗДАНИЮ ФАЙЛА ЗА 30 МИНУТ
================================================================================

Если вы торопитесь и хотите максимально быстро создать рабочий файл,
следуйте этой сокращенной версии инструкции.

ПОЛНОЕ ВРЕМЯ: ~30 минут

================================================================================
ЭТАП 1: СОЗДАНИЕ И СОХРАНЕНИЕ (2 минуты)
================================================================================

1. Откройте Microsoft Excel
2. Создайте новую книгу
3. Сохраните как: ООО_Товарищ_Счета.xlsm
   - Тип файла: "Excel с поддержкой макросов (.xlsm)"
   - Место: C:\Users\YourName\Desktop\ (или другое удобное место)

================================================================================
ЭТАП 2: ПЕРЕИМЕНОВАНИЕ ЛИСТОВ (2 минуты)
================================================================================

1. Right-click на Sheet1 → Rename → "Счет"
2. Right-click на Sheet2 → Rename → "Клиенты"
3. Right-click на Sheet3 → Rename → "Товары"
4. Right-click на пусто в строке листов → Insert Sheet → "Журнал счетов"
5. Insert Sheet → "Реквизиты"
6. Insert Sheet → "Настройки"
7. Insert Sheet → "Справочники"

Удалите пустые листы (если есть).

================================================================================
ЭТАП 3: ЗАПОЛНЕНИЕ РЕКВИЗИТОВ (3 минуты)
================================================================================

На листе "Реквизиты" (в столбцах A и B):

A1: "Название компании"           B1: "ООО "Товарищ""
A2: "ИНН"                         B2: "2534215241"
A3: "КПП"                         B3: "251122101"
A4: "Адрес"                       B4: "690074, Приморский край, г. Владивосток, ул. Снеговая, дом 13Б"
A5: "Телефон"                     B5: "8 (964) 231-70-05"
A6: "Email"                       B6: "(оставить пусто)"
A7: "Директор"                    B7: "(ФИО директора)"
A8: "Главный бухгалтер"           B8: "(ФИО главного бухгалтера)"
A9: "Последний номер счета"       B9: "0"

================================================================================
ЭТАП 4: СОЗДАНИЕ СПРАВОЧНИКА КЛИЕНТОВ (5 минут)
================================================================================

На листе "Клиенты" создайте таблицу:

Строка 1 (Заголовки):
A1: "Название"
B1: "ИНН"
C1: "КПП"
D1: "Адрес"
E1: "Телефон"
F1: "Email"
G1: "Контактное лицо"

Строки 2-10 (Примеры):
A2: "ООО Альфа"              B2: "1234567890"    C2: "123456789"    D2: "Адрес 1"    ...
A3: "ООО Бета"               B3: "0987654321"    C3: "987654321"    D3: "Адрес 2"    ...
A4: "ЗАО Гамма"              B4: "1111111111"    C4: "111111111"    D4: "Адрес 3"    ...

Создайте Excel Table:
- Выделите A1:G10
- Insert → Table
- Назовите: tblКлиенты

================================================================================
ЭТАП 5: СОЗДАНИЕ СПРАВОЧНИКА ТОВАРОВ (5 минут)
================================================================================

На листе "Товары" создайте таблицу:

Строка 1 (Заголовки):
A1: "Артикул"
B1: "Наименование"
C1: "Единица"
D1: "Цена"

Строки 2-10 (Примеры):
A2: "001"          B2: "Ноутбук"           C2: "шт"      D2: "50000"
A3: "002"          B3: "Монитор"           C3: "шт"      D3: "15000"
A4: "003"          B4: "Клавиатура"        C4: "шт"      D4: "2000"
A5: "004"          B5: "Услуга консультации" C5: "услуга" D5: "5000"

Создайте Excel Table:
- Выделите A1:D10
- Insert → Table
- Назовите: tblТовары

================================================================================

================================================================================

A1: "ООО "ТОВАРИЩ""

СТРОКИ 3-4: Реквизиты компании
A3: "ИНН 2534215241      КПП 251122101"

СТРОКА 6-7: Номер и дата

================================================================================
ЭТАП 7: СОЗДАНИЕ ТАБЛИЦЫ ТОВАРОВ (5 минут)
================================================================================

На листе "Счет", строка 15:

Заголовки (с красным фоном #B71C1C, белый текст):
A15: "№"
B15: "Наименование"
C15: "Ед."
D15: "Количество"
E15: "Цена"
F15: "Сумма"

Строки 16-50: Пусто (будут заполняться)

Вставьте формулы:
- A16: =IF(B16="","",ROW()-15)
- F16: =IF(D16="","",D16*E16)
- Скопируйте вниз до 50 строки

Создайте Excel Table (опционально):
- Выделите A15:F50
- Insert → Table
- Назовите: tblТовары

================================================================================

СТРОКА 55: Сумма прописью
A55: "Сумма прописью:"

================================================================================
ЭТАП 9: ДОБАВЛЕНИЕ VBA-КОДА (3 минуты)
================================================================================

1. Alt + F11 (открыть редактор VBA)
2. Insert → Module
3. Скопируйте весь код из файла "2-VBA-Макросы.txt"
4. Вставьте в редактор
5. Ctrl + S (сохранить)
6. Закройте редактор (Alt + Q)

================================================================================
ЭТАП 10: ДОБАВЛЕНИЕ КНОПОК (3 минуты)
================================================================================

На листе "Счет" добавьте кнопки:

1. Developer → Insert → Button (Form Control)
2. Нарисуйте кнопку
3. Установите текст и макрос:

Кнопка 1:
- Текст: "Новый счет"
- Макрос: НовыйСчет
- Положение: H2
- Цвет: Красный (#B71C1C)

Кнопка 2:
- Текст: "Добавить товар"
- Макрос: ДобавитьСтроку
- Положение: H15
- Цвет: Синий

Кнопка 3:
- Текст: "Удалить товар"
- Макрос: УдалитьСтроку
- Положение: I15
- Цвет: Оранжевый

Кнопка 4:
- Текст: "Печать"
- Макрос: Печать
- Положение: H56
- Цвет: Зеленый

Кнопка 5:
- Текст: "Сохранить PDF"
- Макрос: СохранитьPDF
- Положение: I56
- Цвет: Фиолетовый

================================================================================
ЭТАП 11: БЫСТРОЕ ФОРМАТИРОВАНИЕ (2 минуты)
================================================================================

1. Выделите A1:F59
2. Format → Cells
3. Вкладка: Border
4. Выберите: Grid (все границы)
5. OK

Цвета (если время позволяет):
- Красный фон для заголовков: Select → Format → Fill → Red (#B71C1C)
- Белый текст для заголовков: Select → Format → Font → White

================================================================================
ЭТАП 12: НАСТРОЙКА ПЕЧАТИ (2 минуты)
================================================================================

1. Page Layout → Print Area → Set Print Area
2. Выделите A1:F59
3. Page Layout → Page Setup
4. Размер: A4
5. Ориентация: Портретная
6. Поля: Узкие

================================================================================
ЭТАП 13: СОХРАНЕНИЕ (1 минута)
================================================================================

1. Ctrl + S
2. Закройте файл
3. Откройте заново
4. Проверьте, что номер счета сохранился

ГОТОВО! Файл готов к использованию!
