ВОПРОС 1: Как добавить новый лист в таблицу?
================================================================================

ОТВЕТ:
1. Right-click на ярлычок листа (внизу экрана)
2. Select: Insert Sheet
3. Введите имя листа
4. OK

ПРИМЕЧАНИЕ: Если включена защита структуры, этого не получится сделать!

================================================================================
ВОПРОС 2: Как изменить пароль на защиту?
================================================================================

ОТВЕТ:
1. Снимите текущую защиту:
   - Tools → Protect Sheet
   - Введите пароль: Товарищ2024
   - OK

2. Установите новую защиту:
   - Tools → Protect Sheet
   - Введите новый пароль
   - OK

================================================================================
ВОПРОС 3: Как экспортировать счет в PDF?
================================================================================

ОТВЕТ:
Метод 1 - Использовать кнопку "Сохранить PDF":
1. Заполните счет
2. Нажмите кнопку "Сохранить PDF"
3. Файл автоматически сохранится в папку "PDF"

Метод 2 - Стандартный экспорт Excel:
1. File → Export as PDF
2. Выберите папку
3. OK

================================================================================
ВОПРОС 4: Как изменить логотип компании?
================================================================================

ОТВЕТ:
1. Откройте лист "Счет"
2. Right-click на ячейку A1
3. Select: Insert Picture
4. Выберите файл логотипа
5. Отрегулируйте размер (2 см х 2 см)

================================================================================
ВОПРОС 5: Как добавить новый товар в справочник?
================================================================================

ОТВЕТ:
1. Откройте лист "Товары"
2. Перейдите на новую строку (например, А5)
3. Введите:
   - Артикул (например: 005)
   - Наименование
   - Единицу измерения
   - Цену

4. Товар автоматически появится в выпадающем списке на счете

ПРИМЕЧАНИЕ: Если таблица защищена, сначала снимите защиту.

================================================================================
ВОПРОС 6: Как добавить нового клиента?
================================================================================

ОТВЕТ:
1. Откройте лист "Клиенты"
2. Перейдите на новую строку
3. Введите:
   - Название компании
   - ИНН
   - КПП
   - Адрес
   - Телефон
   - Email
   - Контактное лицо

4. Клиент автоматически появится в выпадающем списке на счете

================================================================================
ВОПРОС 7: Почему не работает кнопка "Сохранить PDF"?
================================================================================

ОТВЕТ:
Возможные причины:

1. Счет не заполнен
   → Заполните хотя бы название клиента
   
2. Папка "PDF" не создана
   → Макрос создаст её автоматически при первом использовании
   
3. Макросы отключены
   → Включите их: File → Options → Trust Center → 
     Trust Center Settings → Macro Settings → 
     Enable All Macros

4. Недостаточно прав на запись
   → Проверьте, что у папки есть права на запись

================================================================================
ВОПРОС 8: Как сбросить номер счета на 1?
================================================================================

ОТВЕТ:
1. Откройте лист "Реквизиты"
2. Найдите ячейку B9 (Последний номер счета)
3. Введите: 0
4. OK

Следующий счет будет иметь номер 000001

ВНИМАНИЕ: Будьте осторожны! Это может запутать историю!

================================================================================
ВОПРОС 9: Как изменить формат даты?
================================================================================

ОТВЕТ:
1. Right-click на ячейке E7 (дата счета)
2. Select: Format Cells
3. Вкладка: Numbers
4. Category: Date
5. Выберите нужный формат
6. OK

ПРИМЕРЫ ФОРМАТОВ:
- 23.06.2026
- 23/06/2026
- 23 июня 2026 г.
- Wednesday, June 23, 2026

================================================================================
ВОПРОС 10: Как распечатать несколько счетов сразу?
================================================================================

ОТВЕТ:
К сожалению, встроенной функции нет.
Используйте стандартный способ:

1. Для каждого счета:
   - Заполните данные
   - Нажмите "Печать"
   - Нажмите "Новый счет"

Или используйте PDF:
1. Заполните счет
2. Нажмите "Сохранить PDF"
3. Все PDF-файлы можно распечатать как группу

================================================================================
ВОПРОС 11: Как изменить единицы измерения?
================================================================================

ОТВЕТ:
1. Откройте лист "Счет"
2. Выделите ячейку C16:C50
3. Data → Validity
4. Измените список Source на ваш:
   новая_единица1;новая_единица2;...
5. OK

Текущий список:
шт;м²;м³;кг;г;т;л;рулон;лист;упаковка;мешок;комплект;услуга

================================================================================
ВОПРОС 12: Как добавить НДС в счет?
================================================================================

ОТВЕТ:
1. На листе "Счет" вставьте новую строку перед "Всего к оплате"
2. Введите: "НДС (18%):"
3. В соседнюю ячейку введите формулу: =F52*0.18
4. Измените формулу в F54: =F52+F53 (где F53 - НДС)

ПРИМЕЧАНИЕ: Согласно заданию, НДС не используется. 
Это решение для будущего, если потребуется.

================================================================================
ВОПРОС 13: Как сделать резервную копию файла?
================================================================================

ОТВЕТ:
1. Закройте файл
2. Скопируйте ООО_Товарищ_Счета.xlsm
3. Переименуйте копию: ООО_Товарищ_Счета_Резервная_2026-06-23.xlsm
4. Сохраните в безопасном месте (облако, внешний диск)

РЕКОМЕНДАЦИЯ: Делайте резервную копию каждый день!

================================================================================
ВОПРОС 14: Как отменить последнее действие?
================================================================================

ОТВЕТ:
Используйте стандартный Excel:
- Ctrl + Z (отменить)
- Ctrl + Y (повторить)

Или используйте меню:
- Edit → Undo
- Edit → Redo

ПРИМЕЧАНИЕ: Если вы уже создали новый счет, 
отмена может не восстановить старые данные.

================================================================================
ВОПРОС 15: Как изменить цвет компании с красного на другой?
================================================================================

ОТВЕТ:
1. Выделите все ячейки с красным цветом (заголовки, итоги)
2. Format → Cells
3. Вкладка: Fill
4. Выберите нужный цвет
5. OK

ТЕКУЩИЙ ЦВЕТ: RGB(183, 28, 28) = #B71C1C

================================================================================
ВОПРОС 16: Как добавить подпись директора на печать?
================================================================================

ОТВЕТ:
1. На листе "Счет" перейдите на строку 57
2. В ячейке B57 (пустая строка для подписи)
3. Вместо печати подпишите вручную перед отправкой
4. Или используйте электронную подпись (Advanced Digital Signature)

================================================================================
ВОПРОС 17: Как экспортировать все счета за месяц?
================================================================================

ОТВЕТ:
1. Откройте лист "Журнал счетов"
2. Примените фильтр: Data → AutoFilter
3. Отфильтруйте по дате (например, июнь 2026)
4. Выделите все видимые записи
5. Скопируйте
6. Вставьте в отчет (Word, Google Docs и т.д.)

Или используйте PDF:
1. Для каждого счета нажимайте "Сохранить PDF"
2. Все PDF-файлы будут в папке "PDF"
3. Архивируйте и сохраняйте

ВОПРОС 19: Как защитить файл от открытия паролем?
================================================================================

ОТВЕТ:
1. File → Save As
2. Нажмите кнопку "Tools"
3. Select: General Options
4. Введите пароль в "Password to open"
5. Подтвердите пароль
6. Save

ВНИМАНИЕ: Если забудете пароль, файл не откроется!

================================================================================
ВОПРОС 20: Что если я случайно удалил важную формулу?
================================================================================

ОТВЕТ:
Немедленно нажмите: Ctrl + Z (отменить)

Если уже много действий прошло:
1. Закройте файл БЕЗ сохранения
2. Откройте снова
3. Все вернется в предыдущее состояние

СОВЕТ: Включите автосохранение: File → Options → Save →
       AutoRecover every N minutes

ТЕХНИЧЕСКАЯ ПОДДЕРЖКА
================================================================================

Если документ не работает:

1. Проверьте версию Excel:
   - File → Account → About
   - Должна быть версия 2016 или выше

2. Включите макросы:
   - File → Options → Trust Center → 
     Trust Center Settings → Macro Settings → 
     Enable All Macros

3. Перезагрузитесь:
   - Закройте все приложения
   - Перезагрузите компьютер
   - Откройте файл заново

4. Проверьте антивирус:
   - Убедитесь, что антивирус не блокирует макросы
   - Добавьте файл в исключения антивируса

5. Пересоздайте файл:
   - Если все еще не работает, начните заново
   - Копируйте данные, но переделайте макросы
