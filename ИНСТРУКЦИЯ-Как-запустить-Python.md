ШАГ 1: УСТАНОВКА PYTHON (если еще не установлен)
================================================================================

ВАРИАНТ 1 - Скачать с официального сайта (РЕКОМЕНДУЕТСЯ):

1. Перейдите: https://www.python.org/downloads/
2. Нажмите кнопку "Download Python 3.11" (или более новую версию)
3. Дождитесь загрузки файла (примерно 30 МБ)
4. Откройте загруженный файл (python-3.11.x-amd64.exe или похожий)
5. В окне установки:
   ☑ Установите галочку "Add Python to PATH" (ВАЖНО!)
   ☑ Нажмите "Install Now"
6. Дождитесь завершения установки
7. Нажмите "Close"

ПРОВЕРКА:
- Откройте командную строку (Win+R → cmd → Enter)
- Введите: python --version
- Должно показать: Python 3.11.x (или похожее)

ВАРИАНТ 2 - Через Microsoft Store (для Windows 10/11):

1. Откройте Microsoft Store
2. Найдите: "Python 3.11"
3. Нажмите "Get"
4. Дождитесь установки
5. Проверьте как выше

================================================================================
ШАГ 2: УСТАНОВКА ЗАВИСИМОСТЕЙ
================================================================================

Python нужна библиотека "openpyxl" для работы с Excel.

1. Откройте командную строку (Win+R → cmd → Enter)

2. Введите команду (скопируйте полностью):

   pip install openpyxl

3. Нажмите Enter

4. Дождитесь завершения (должно написать "Successfully installed")

ГОТОВО! Все зависимости установлены.

ШАГ 3: СКАЧИВАНИЕ СКРИПТА
================================================================================

ВАРИАНТ 1 - Клонирование репозитория (РЕКОМЕНДУЕТСЯ для обновлений):

1. Откройте командную строку (Win+R → cmd → Enter)

2. Выберите папку, где хотите сохранить проект:
   cd C:\Users\YourName\Desktop
   (или другой путь)

3. Скопируйте и введите команду:

Теперь у вас есть папка "invoice-system" со всеми файлами!

2. Нажмите кнопку "Code" (зеленая кнопка справа)

6. Нажмите "Extract All"

1. Откройте любой текстовый редактор (Notepad)

ШАГ 6: ДОБАВЛЕНИЕ ПОЛНЫХ VBA-МАКРОСОВ
================================================================================

Скрипт создает базовый файл с простыми макросами.
Для полного функционала нужно добавить полный VBA-код.

1. Откройте файл ООО_Товарищ_Счета.xlsm в Excel

2. Нажмите Alt + F11 (откроется редактор VBA)

3. В левой панели кликните правой кнопкой на "Modules"

4. Select: Insert → Module

5. Скопируйте весь код из файла "2-VBA-Макросы.txt"
   (из репозитория на GitHub)

6. Вставьте в редактор VBA

7. Нажмите Ctrl + S (сохранить)

8. Закройте редактор VBA (Alt + Q)

9. Сохраните файл Excel (Ctrl + S)

ГОТОВО! Теперь все макросы работают!

РЕШЕНИЕ ТИПИЧНЫХ ПРОБЛЕМ
================================================================================

❌ ОШИБКА: "Python is not recognized"
✅ РЕШЕНИЕ:
   - Python не установлен или не добавлен в PATH
   - Переустановите Python и установите галочку "Add Python to PATH"
   - Перезагрузитесь после установки

❌ ОШИБКА: "No module named 'openpyxl'"
✅ РЕШЕНИЕ:
   - Введите в командной строке: pip install openpyxl
   - Дождитесь установки

❌ ОШИБКА: "Access denied" или "Permission denied"
✅ РЕШЕНИЕ:
   - Запустите командную строку "от имени администратора"
   - Переместите папку в Documents ил�� Desktop
   - Проверьте, что папка не защищена от записи

❌ ОШИБКА: "File not found" или "No such file or directory"
✅ РЕШЕНИЕ:
   - Проверьте, что вы находитесь в правильной папке (cd ...)
   - Проверьте, что файл create_invoice_system.py в этой папке
   - Скопируйте полный путь из адресной строки: cd "полный путь"

❌ ПРОБЛЕМА: Файл создался, но Excel его не открывает
✅ РЕШЕНИЕ:
   - Файл может быть поврежден
   - Запустите скрипт еще раз
   - Проверьте, что установлена последняя версия Python

❌ ПРОБЛЕМА: Макросы не работают
✅ РЕШЕНИЕ:
   - Вы не добавили VBA-код (ШАГ 6)
   - Или макросы отключены в Excel
   - Или нужно включить содержимое при открытии
